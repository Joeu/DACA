===== PROBLEM CONTROLLER =====


===== GET =====

GET /problem

  • Descrição - Retorna todos os problemas cadastrados no sistema.
  • Parametros - Não possui.
  • Entrada e Saida
     Entrada: http://localhost:8080/bootwildfly/problem
     Saída:
    {
     {"description": "Programa que calcula a media de um numero", "name": "Prob1", "status": "nao submetido", "tip": "teste"}
     {"description": "Programa que calcula o fatorial de um numero", "name": "Prob2", "status": "nao submetido", "tip": "teste2"}
    }
  • Mensagem de Retorno - Problemas retornados com sucesso.

GET /problem/id

  • Descrição - Retorna um problema especifico cadastrado no sistema.
  • Parametros - id do problema a ser procurado
  • Entrada e Saida
     Entrada: http://localhost:8080/bootwildfly/problem/1
     Saída:
    {
     {"description": "Programa que calcula a media de um numero", "name": "Prob1", "status": "nao submetido", "tip": "teste"}
    }
  • Mensagem de Retorno - Problema retornado com sucesso.

GET /problem/id/solution

  • Descrição - Retorna as soluções submetidas para o referido problema.
  • Parametros - id do problema.
  • Entrada e Saida
     Entrada: http://localhost:8080/bootwildfly/problem/1/solution
     Saída:
    {
     {"iduser": 1, "idProblem": 1, "content": "print 'Hello World'", "status": "Failed"}
     {"iduser": 1, "idProblem": 1, "content": "print 'Teste'", "status": "Pass"}
    }
  • Mensagem de Retorno - Soluções retornadas com sucesso.

GET /problem/id/solution/id

  • Descrição - Retorna uma solução especifica para o problema.
  • Parametros - id do problema e id da solução.
  • Entrada e Saida
     Entrada: http://localhost:8080/bootwildfly/problem/1/solution/1
     Saída:
    {
     {"iduser": 1, "idProblem": 1, "content": "print 'Hello World'", "status": "Failed"}
    }
  • Mensagem de Retorno - Solução retornada com sucesso.

GET /problem/id/test

  • Descrição - Retorna todos os testes de um problema.
  • Parametros - id do problema
  • Entrada e Saida
     Entrada: http://localhost:8080/bootwildfly/problem/1/test
     Saída:
    {
     {"id": 1, "name": "test problem1 1", "input": "2", "output": "4"}
     {"id": 2, "name": "test problem1 2", "input": "4", "output": "6"}
    }
  • Mensagem de Retorno - Testes da solução retornados com sucesso.

GET /problem/id/test/id

  • Descrição - Retorna todos um teste especifico de um problema.
  • Parametros - id do problema e id do teste
  • Entrada e Saida
     Entrada: http://localhost:8080/bootwildfly/problem/1/test/1
     Saída:
    {
     {"id": 1, "name": "test problem1 1", "input": "2", "output": "4"}
    }
  • Mensagem de Retorno - Teste da solução retornado com sucesso.

===== POST =====

POST /problem/

  • Descrição - Cria um novo problema.
  • Parametros - Objeto do tipo Problem
  • Entrada e Saida
     Entrada: http://localhost:8080/bootwildfly/problem/
  • Mensagem de Retorno - Problema criado com sucesso.

POST /problem/id/solution

  • Descrição - Usuário posta uma solução para um problema.
  • Parametros - Objeto do Submission e o id problema
  • Entrada e Saida
     Entrada: http://localhost:8080/bootwildfly/problem/1/solution
  • Mensagem de Retorno - Solução postada com sucesso.

POST /problem/id/test

  • Descrição - Cria um teste para o problema.
  • Parametros - Objeto do tipo Test e id do problema.
  • Entrada e Saida
     Entrada: http://localhost:8080/bootwildfly/problem/1/test
  • Mensagem de Retorno - Teste criado com sucesso.
